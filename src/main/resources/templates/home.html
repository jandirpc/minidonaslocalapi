Bienvenido a su tienda virtual de Donas de DoniBites

Endpoints disponibles para leer datos

  • Lista de usuarios
  • Roles de usuarios
  • Lista de categorías
  • Lista de proveedores
  • Lista de productos
  • Lista de pedidos
  • Lista de facturas

Documentación Endpoints

Usuarios

	
		GET (Listar todos los usuarios)
		curl "http://localhost:8080/minidonas/users"
		---------------------------------------------------------------------------------------------------------------------------------
		GET (Buscar un usuario por id, regresa un usuario)
		curl "http://localhost:8080/minidonas/users/find/{id}"
		---------------------------------------------------------------------------------------------------------------------------------
		POST (Crear nuevo usuario, regresa un usuario)
		curl -d @ejemplousuario.json "http://localhost:8080/minidonas/users/newUser"  -H "Content-Type:application/json" 
		---------------------------------------------------------------------------------------------------------------------------------
		PUT (Actualizar un usuario por id, regresa un usuario)
		curl -X PUT -d @ejemplousuario.json "http://localhost:8080/minidonas/users/update/{id}"  -H "Content-Type:application/json"
		---------------------------------------------------------------------------------------------------------------------------------
		Desativar usuario (Donde id es el identificador del usuario a desactivar, regresa true)
		curl "http://localhost:8080/minidonas/users/delete/{id}"
		---------------------------------------------------------------------------------------------------------------------------------
		
		Ejemplo de un payload para usuario (ejemplousuario.json)
		
		{
			"username": "usuario11",
			"password": "contrausuario11",
			"email": "usuario11@me.com",
			"rol": 	{
						"id":1
					},
			"completeName": "Juan Medica",
			"address": "Villa Libertad Casa 88",
			"phone": "+50586448855",
			"status": "activo"
		}
		---------------------------------------------------------------------------------------------------------------------------------
